
## A. Goal Of App

    1. Account for money spent
    2. categorise spending and incoming in appropriate categories later to seen in a chart format for analysis

## A.1 - Specs

    1️⃣ Key Requirements Recap (from your spec)

    CRUD for transactions (credit/debit)
    Dynamic categories (create on the go)
    Multiple bank accounts
    Time-based aggregation (monthly / quarterly / yearly)
    Charts & summaries
    Secure balance display (masked, tap to reveal)
    Future extensibility

    This clearly maps to a full-stack app with analytics.

## B. Base Features

    1. Take inputs for both credit and debit with an option for defaults categores to go with it, or create a category on the go, to be associated with the current entry, also associate the bank account if multiple accounts exist
    2. show monthly, quarterly and yearly data summarise in a summary tab to have a quick look at health of expenses
    3. input current balance of each bank account and display star in place of each digit to be shown value when touched
